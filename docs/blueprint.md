# **App Name**: StockPilot

## Core Features:

- Inventory Management UI: Simple and clear user interface for managing inventory (tools, materials, and PPE).
- Employee Assignment: Employee assignment functionality to assign inventory items to specific employees.
- Dashboard Reporting: Dashboard view providing a summary of current stock levels and assigned items.
- AI-Powered Discrepancy Detection: Tool that automatically identifies discrepancies in inventory based on assignment records and current stock, generating alerts for review.

## Style Guidelines:

- Primary color: A muted blue (#6699CC) evoking a sense of trust and reliability, in line with the importance of careful inventory management.
- Background color: A light, desaturated blue (#E6F0F7), offering a clean and professional backdrop.
- Accent color: A warm yellow (#FFDA63), creating contrast and drawing the user's attention to key actions or alerts.
- Body and headline font: 'Inter' (sans-serif) for a clean, modern, and highly readable interface. 
- Simple, clear icons inspired by Google's Material Design to represent inventory items and actions.
- Clean and organized layout with clear visual hierarchy for easy navigation and information access.
- Subtle transitions and animations to provide feedback and guide the user through the application.
